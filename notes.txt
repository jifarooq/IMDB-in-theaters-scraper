SETUP:

1. use `go get -u {dependency}` for each dependency you don’t already have locally
2. type `go mod tidy` to populate your go.sum file
3. (optional) to use vender folder, type “go mod vendor” (but generally don’t have to do this in go 1.13 and higher)


When building, must set Linux as OS. Steps:
  GOOS=linux go build imdbscrape.go
  zip imdbscrape.zip imdbscrape
